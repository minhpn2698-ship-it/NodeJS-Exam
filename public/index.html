🎯 API Quản lý Khách hàng

Server đang chạy thành công! Sử dụng các endpoint dưới đây:

Khách hàng (Customers)

GET /api/customers - Lấy tất cả khách hàng
GET /api/customers/:id - Lấy khách hàng theo ID
GET /api/customers/search?q=keyword - Tìm kiếm khách hàng
POST /api/customers - Tạo khách hàng mới
PUT /api/customers/:id - Cập nhật khách hàng
DELETE /api/customers/:id - Xóa khách hàng

Loại Khách hàng (Customer Types)

GET /api/customer-type - Lấy tất cả loại
GET /api/customer-type/:id - Lấy loại theo ID
POST /api/customer-type - Tạo loại mới
PUT /api/customer-type/:id - Cập nhật loại
DELETE /api/customer-type/:id - Xóa loại
